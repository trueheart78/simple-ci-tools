# Simple CLI Tools

Because I am lazy

## Running Tests

```ruby
bundle install
bundle exec rake test
```

## Adding A Binary

```sh
touch bin/awesome_cmd
chmod +x bin/awesome_cmd
```

## Adding to Your path

Add the full `bin/` path to your `PATH`

```
export PATH="$PATH:/Users/jmills/ruby/simple-cli-tools/bin"
```
